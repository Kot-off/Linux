#Linux

##Место на дисках

###Посмотреть место на дисках

```
df -h
```

###Показывает сколько занимают файлы в определенных директориях

```
du -hd1
du - sh /tmp
```

###Inode - структуры которые хранят некую служебную информацию о файлах в Linux.

```
df -i
```

###Общее состояние жестких дисков

```
lsblk
```
